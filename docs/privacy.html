Privacy Policy

Last updated: June 2026  ·  WikiRoll Discord Bot

WikiRoll collects the minimum data needed to run the bot. We do not sell, share, or monetize your data in any way.

1. What We Collect

WikiRoll stores the following data in a local SQLite database on our server:

  • Discord User ID — to associate character ownership, wishlists, and cooldowns with your account
  • Discord Server (Guild) ID — to scope ownership and settings per server
  • Character records — name, source wiki URL, thumbnail URL, and Wikipedia/Fandom page ID
  • Ownership records — which user in which server owns which character
  • Wishlist entries — character names or sources you've wishlisted
  • Roll cooldowns — timestamp of your last roll per server
  • User-submitted image URLs — Discord CDN URLs you provide via /submitimage
  • Trade records — active and completed trades between users

We do not collect: usernames, email addresses, message content (beyond command arguments), IP addresses, or any other personal information.

2. How We Use It

All data collected is used solely to operate the bot's core features:

  • Tracking character ownership within each server
  • Enforcing roll cooldowns
  • Sending DM notifications when wishlisted characters are rolled
  • Processing trades between users

3. Data Retention

Your data is retained for as long as you use the bot. You can:

  • Remove individual characters from your collection with /remove
  • Clear your wishlist with /wishlist remove
  • Request full deletion of your data by opening a GitHub Issue

If the bot is removed from a server, that server's data may be retained in the database indefinitely. Server admins can contact us to request deletion.

4. Data Sharing

We do not sell, rent, or share your data with third parties. The bot fetches character data from public Wikipedia and Fandom APIs — your Discord information is never sent to those services.

5. Security

Data is stored in a SQLite database on a self-hosted server. Access is restricted to the server administrator. We take reasonable precautions but cannot guarantee absolute security.

6. Children

WikiRoll does not knowingly collect data from users under 13. If you believe a child has used the bot, please contact us for data removal.

7. Changes

We may update this policy from time to time. Continued use of the bot after changes constitutes acceptance of the new policy.

8. Contact

For privacy requests or questions, open an issue at github.com/Hackatoan/wikiroll/issues or visit hackatoa.com.
